# Movie Mogul 🎬

A turn-based strategy game where you build a Hollywood empire from a tiny 1970s indie studio into a legendary entertainment conglomerate.

## Play

**In Claude:** Open `MovieMogul.jsx` as a React artifact — it's a single self-contained component.

**On the web:** Deploy to Vercel or run locally:

```bash
npm install
npm run dev
```

## How to Play

1. **Name your studio** and start in 1970 with $2M
2. **Develop films** — choose scripts, set budgets, assign talent
3. **End turns** to advance time (each turn = 1 quarter)
4. **Grow your empire** through box office hits, awards, and smart investments
5. **Unlock milestones** to become a Hollywood Legend

## Game Features

- 🎥 Full film production pipeline (development → production → post → release)
- ⭐ Talent management — sign directors, actors, and writers
- 📊 Realistic box office simulation with quality-driven returns
- 🏆 Awards system and prestige tracking
- 📺 Launch your own streaming platform (2005+)
- 🎭 Era progression: New Hollywood → Blockbuster → Digital → Streaming Wars
- 📈 Genre trends, market events, and industry dynamics

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Import the repo on [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — no config needed
4. Deploy!

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Recharts for data visualization
- Zero external APIs — runs entirely client-side
